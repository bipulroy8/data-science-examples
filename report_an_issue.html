Data Science Examples: How to report an issue


Send email to h2ostream, or visit the h2ostream web page.

(h2ostream is the H2O open source community forum.)


Including as much of the following information as you can helps a lot:

  • H2O version information
  • General environment (OS type)
  • A link to the example you are trying
  • The specific tab you are trying
  • Are you using the H2O Flow web UI? If so, include a screenshot.
  • Console output with the error message in context
  • H2O logs
